# Slack App Example

Código referente a um post no [dev.to](https://dev.to/) sobre criação de apps a partir da Web API do Slack. Para acessá-lo, clique [neste link](https://dev.to/lukehxh/criar-um-slack-app-para-impulsionar-seu-time-33dh).

## Rodar

No arquivo `src/services/slack.js`, insira o token de sua aplicação slack para funcionar.

```
npm install
npm run dev
```

Ou em versão de produção:

```
npm start
```
